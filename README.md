# 🚆 Railway Seat Booking System for Visually Impaired

A voice-assisted command-line railway ticket booking system designed to help visually impaired users navigate the booking process independently.

---

## 🎯 Problem Statement

Traditional railway booking systems are not accessible for visually impaired users.  
This project aims to simplify the process using voice guidance and structured interaction.

---

## 💡 Features

- 🔊 Voice guidance using text-to-speech (pyttsx3)
- 📍 Station selection (Chennai to multiple destinations)
- 👥 Passenger count input
- 💺 Class selection (AC / Sleeper)
- 💰 Automatic fare calculation
- 💳 Simulated payment system with OTP verification
- ⭐ Feedback system at the end

---

## 🛠️ Tech Stack

- Python  
- pyttsx3 (Text-to-Speech)

---

## ▶️ How to Run

1. Install dependencies:
   pip install pyttsx3

2. Run the program:
   python railway_booking.py


---

## 📌 How It Works

- User interacts via keyboard input
- System provides continuous voice guidance
- Ticket price is calculated based on user choices
- Payment flow is simulated
- Booking confirmation and feedback collected
